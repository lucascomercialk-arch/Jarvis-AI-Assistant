 Jarvis AI Assistant

Assistente pessoal inteligente desenvolvido para explorar conceitos de Inteligência Artificial, automação e interação homem-máquina.

O projeto tem como objetivo evoluir para uma plataforma pessoal capaz de interagir por voz, executar automações, armazenar contexto e integrar dispositivos e serviços externos.

##  Tecnologias Utilizadas

- Python
- Flask
- SQLite
- Text-to-Speech (TTS)
- Three.js
- Integração com APIs de IA
- HTML, CSS e JavaScript

## Funcionalidades Atuais

-  Responde perguntas e interage com o usuário
-  Consulta informações na internet
-  Controle por voz
-  Armazenamento de informações e preferências
-  Interface com animações e elementos visuais
-  Manipulação de objetos e formas 3D utilizando webcam
-  Estrutura preparada para integração com APIs de IA

##  Arquitetura do Projeto

O sistema é composto por:

- Backend em Flask responsável pela lógica principal
- Banco de dados SQLite para armazenamento local
- Interface interativa utilizando Three.js
- Sistema de voz utilizando TTS
- Módulo preparado para integração com modelos de linguagem (LLMs)
